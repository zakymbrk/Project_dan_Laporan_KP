ℹ️ Informasi Tambahan

Format Sistem (18 Kolom):

  • Username *, Nama Lengkap *, Password *, NIS, NISN
  • Jenis Kelamin (L/P), Tempat Lahir, Tanggal Lahir (YYYY-MM-DD)
  • Kelas *, Jurusan, Telepon, Alamat, Email
  • Periode, Tanggal Mulai/Selesai (YYYY-MM-DD), Nama DUDI, Status Pengajuan

Template Akun Siswa (6 Kolom):

  • Username *, Nama Lengkap *, Password *, Kelas *
  • Email, Telepon

Catatan: Pastikan file Template.xlsx berada di folder assets/ sebelum menggunakan fitur konversi.
